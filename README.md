# LockRay

Behavioral diff analysis for dependency PRs.

> Dependabot tells you *what* changed; LockRay tells you whether the change *behaves* suspiciously.

**Status:** Pre-v0.1 — currently building M1 (skeleton + lockfile parsers). See [design spec](docs/specs/2026-04-20-lockray-design.md).

## Development

```bash
nvm use         # Node 22
npm install
npm run build
npm test
```
